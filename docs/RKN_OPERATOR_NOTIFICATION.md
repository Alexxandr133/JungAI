### Цели обработки персональных данных *

В форме можно добавить **несколько целей** (кнопка «+»). Ниже — типовой набор для JungAI; каждую цель вносите отдельной строкой с привязкой к категориям ПДн, субъектам и основаниям (см. блоки ниже).

| № | Цель обработки ПД (формулировка для формы) |
|---|---------------------------------------------|
| 1 | Регистрация, идентификация и аутентификация пользователей на платформе JungAI (`https://jung-ai.ru`) |
| 2 | Исполнение пользовательского соглашения / публичной оферты: предоставление функционала (профиль, чат, календарь, CRM клиентов, рабочая область, публикации, исследовательские модули) |
| 3 | Оказание услуг психологической поддержки через платформу: ведение записей о клиентах, сессиях, сновидениях, документов, переписки между психологом и клиентом |
| 4 | Проведение видео- и аудиовстреч (видеозвонки) в рамках запланированных событий |
| 5 | Обработка обращений в техническую поддержку и верификация профессионального статуса психологов |
| 6 | Направление сервисных уведомлений (подтверждение e-mail, сброс пароля, уведомления о встречах) |
| 7 | Обеспечение информационной безопасности, предотвращение злоупотреблений, ведение технических логов |
| 8 | Улучшение качества сервиса (при наличии аналитики — только при отдельном уведомлении пользователя) |
| 9 | Использование функций ИИ-ассистента для анализа текстов пользователя в заявленных целях продукта |
| 10 | Публичная запись на приём через страницу календаря (данные заявителя без учётной записи) |
| 11 | Соблюдение требований законодательства РФ (ответы на законные запросы госорганов) |

| Иные пункты | По ситуации (ГИС-услуги, суд, жизненно важные интересы) — для JungAI обычно **не основные** |
